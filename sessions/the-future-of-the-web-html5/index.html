Session

The Future of the Web: HTML5

The latest version of HTML may not be completed for some time, but there are parts that are available in the latest version of browsers, and shims to make older browsers support them as well.  There are many new elements, including those that provide more semantic meaning to common <div> elements (like header, footer, etc).

 A new <canvas> can be used for drawing, while the <audio> and <video> provide new multimedia capabilities.  Learn about support for HTML 5 in major browsers, and see what can be done with the new standard.
