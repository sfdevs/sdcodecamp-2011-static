Welcome!

South Dakota Code Camp is an event hosted by the Sioux Falls Developers Group and South Dakota .Net User Group with the sponsorship and support from and many area businesses.

Most of you visiting may be wonder "Whats a code camp?" Here is the basic idea.

A code camp IS:

  • Community Driven 
  • Free to attendees
  • Relevant to community members
  • NEVER during working hours (usually on Saturdays)

A code camp IS NOT

  • Product Demonstration/Sales
  • Corporate Marketing

If you're passionate about programming, design, or something we haven't thought of that you'd love to teach others about, you should submit a session.
